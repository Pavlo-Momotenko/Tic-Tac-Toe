## Launching

1. Clone this repository.
2. `cd` into this directory.
3. `\venv\Scripts\activate.bat`
4. `python ttt.py`
5. Enjoy!

or 

1. Clone this repository.
2. Open compiled `TicTacToe.exe` in TTT folder.
